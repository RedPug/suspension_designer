# Suspension Designer
A 3D suspension design software, created for Anteater Formula Racing and the University of California, Irvine.

## Current features:
- Add and modify suspension nodes
- Create node groups to define connected bodies
- Design suspenion geometry to exist along reference geometry
- Solve 3D geometry fast and accurately
  - Solve for and apply constraints using custom variables
    - Measure displacement in a direction (i.e. heave, steering rack movement)
    - Measure distance between nodes (i.e. spring length)


## Planned Features:
- Force solving
- Full vehicle dynamic simulation
  - Mass info + tire data + 4 wheel simulation
  - Will come in stages
    1. Track height profile for a straight track
    2. Track height data for a full 3D simulation
